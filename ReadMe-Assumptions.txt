Assumptions

1.A user can have many followers.
2.A user can follow back his followers.
3.A user can see his tweets.
4.A user can see tweets of all the people he is following.
5.A user can be a follower visa vesa.
6.A user cannot see tweets of people he is not following.
7.A tweet is displayed even if it exceeds the number of characters, 
with the maximum number of characters allowed (140).
8.The files already exist.
9.The files are saved on current directory "Files" folder.
10. There is no user or follower with the name "follows".
